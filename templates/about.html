Document Scanner Web Application

This web application allows you to extract structured information from document images using advanced computer vision and natural language processing techniques. Perfect for processing business cards and extracting useful information.

How to Use

Step 1: Upload Your Document

  • Click "Choose File" to upload your document image
  • Select your preferred OCR method: Pytesseract (traditional), Qwen2 AI (advanced), or Azure Document Intelligence (cloud-based)
  • If using Qwen2, make sure to load the model first

Step 2: Adjust Document Boundaries

  • The system will automatically detect document corners
  • You can drag the corner points to adjust if needed
  • Click "Transform" to correct the perspective and enhance the image

Step 3: View Extracted Information

  • The system extracts and categorizes information into key entities
  • Entity types include: NAME, ORGANIZATION, DESIGNATION, PHONE, EMAIL, and WEBSITE
  • The processed image shows bounding boxes around detected text

Technologies

Computer Vision

  • OpenCV - Document detection and image processing
  • Pytesseract - Optical Character Recognition

Natural Language Processing

  • SpaCy - Named Entity Recognition
  • Qwen2-VL - Advanced AI for integrated image and text understanding
Note: For best results, ensure your document has good lighting and is free from excessive glare or shadows. The application works best with clearly printed text.
